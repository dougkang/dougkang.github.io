Yet Another Hello World

09 Apr 2013

I’ve been meaning to use jekyll for a while now, but I only now was able to migrate my blog to it. So far, I’m liking it. By default, it uses markdown, which comes naturally to me thanks to github. The workflow for creating posts is more akin to the workflows that I am used to (e.g. writing in plaintext in a console, pushing stuff to git) and the flexibility is pretty amazing.

I’m using this as a chance to revive my thrice-dead blog - hopefully I can make it live longer than last time. I’ve been doing alot of work getting my spark cluster at home working and I find it quite nice, though some things are not quite so nice - again, more on this later.

Update: I was able to get the post-receive hook working - nice!
